### Introducción
----
Este repositorio tiene el objetivo de aplicar el diseño orientado a objetos sobre el sistema de una pequeña productora audiovisual para digitalizar la gestión de sus proyectos de video.  

Se entregaran los Indices correspondientes para anexos, diagramas UML y Herammientas Agile de cada tipo, entre los cuales contendran:

- Anexos, una introducción al diseño que se aplica para el proyecto con requisitos funcionales y no funcionales, casos de uso, un boceto para las clases del sistema, asi como el método en que se aplicaron mejoras al mismo mediante la aplicación de los principios SOLID.   
Este tambien incluye la version final del diagrama de clases propuesto, y correcciones aplicables a problemas del mismo basados en un analisis a traves de patrones de diseño.

- Diagramas UML, diagramas que representan cada caso de uso, sus posibles escenarios, actividades y secuencias, un boceto para las clases del sistema, asi como las mejoras al mismo mediante la aplicación de los principios SOLID y la version final del diagrama de clases del sistema, con unos diagramas que se basan en problemas detectados del mismo en donde se podrian aplicar patrones de diseño.

- Herramientas Agile, que contiene tarjetas CRC, el análisis a una entrevista JAD y matrices CLAE acorde a los casos de uso del sistema.
